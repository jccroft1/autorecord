# Auto Record

Automatically play music though Sonos automatically. 

## Installation 

Required 
- OpenCV https://github.com/hybridgroup/gocv 
- Docker 

## Usage 

Download a release from the tags. 

Export API Key with key "AR_API_KEY" to environment. 

Example 
```bash 
export AR_API_KEY=ABC123
```
